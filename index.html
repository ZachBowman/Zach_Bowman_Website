About Me
Designing and developing innovative games has been a passion of mine since 1995. I majored in computer science at Millersville University, and I've continued to constantly learn new tools, skills, and engines since. In 2006, I co-founded Nightmare Games (now Cricketheads) with my best friend and long-time creative partner. I've been hard at work creating the next generation of innovative games since.

I started working with the web and HTML in the late 90's to support my other projects, but over time, web design and development have become an entirely new pursuit for me. Since those early pages, I've learned CSS, Javascript, jQuery, and a host of other cool technologies. I intend to bridge the gap between these two worlds by producing awe-inspiring web games.

The projects I'm creating today aren't just self-contained ideas. These are the foundations and building blocks of larger, more complex systems I've planned for the future. My goal is to create unique and immersive worlds to explore and then bring as many people into those worlds as possible.

Completed Projects



Word Wars

This was an idea for a supercharged, scrabble-like spinoff game I pitched, and then assembled a team to help develop. Working together, we used jQuery UI, a dictionary API, and data binding we wrote ourselves to make an html web page into a fully interactive board game (using no game engines or canvas tags). It supports up to four players, and features multiple board shapes, dictionary definitions, and a fully functional validation and scoring system.
