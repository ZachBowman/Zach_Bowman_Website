Word Wars

This was an idea for a supercharged, scrabble-like spinoff game I pitched, and then assembled a team to help develop. Working together, we used jQuery UI, a dictionary API, and data binding we wrote ourselves to make an html web page into a fully interactive board game (using no game engines or canvas tags). It supports up to four players, and features multiple board shapes, dictionary definitions, and a fully functional validation and scoring system.
